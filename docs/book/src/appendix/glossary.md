# Glossary

- `Thread`: persisted conversation + state history.
- `Run`: one execution attempt over a thread.
- `RunContext`: mutable workspace for a run.
- `Patch`: ordered list of state `Op` operations.
- `TrackedPatch`: patch plus metadata for traceability.
- `ThreadChangeSet`: append payload persisted to storage.
- `AgentOs`: orchestration layer for registries and run prep.
- `AgentEvent`: canonical runtime stream event.
